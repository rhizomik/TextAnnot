This is a Web application developed in the context of the research project Processing and cataloging of a written corpus in ELE produced by Chinese students, ECCOTEELE (FFI2016-80280-R) coordinated from Universitat de Lleida.

The aim of this application is to facilitate developing and cataloging a corpus of texts produced by Chinese University students of Spanish as a foreign language (ELE).

Source code available from GitHub:

  • Client: Angular application consuming server API.
  • Server: Spring Boot application providing REST API.
